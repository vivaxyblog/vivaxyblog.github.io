# [vivaxyblog.github.io](https://vivaxyblog.github.io)

## Rules

### Headers

- Use `h1` as post header. `h1` cannot be included in the post content, and `h2` ~ `h5` are acceptable.

### Update posts

- Run `npm run build`.

## Install Dependencies

```sh
npm install
```

## Test Locally

```sh
npm run dev
```

## Build Locally

```sh
npm run build
```
